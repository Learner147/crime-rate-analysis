## Project Overview

This project analyzes crime rate data using Python and data science techniques. The study includes data cleaning, exploratory data analysis (EDA), statistical testing, machine learning models, and clustering methods to uncover crime trends and the influence of socio-economic factors.

## Objectives

- Analyze crime patterns and trends.
- Study the relationship between socio-economic factors and crime rates.
- Visualize crime distributions using charts and graphs.
- Build predictive machine learning models.
- Identify regions with similar crime characteristics through clustering.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Methods

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Feature Engineering
- Linear Regression
- Random Forest Regression
- K-Means Clustering

## Results

The analysis highlights important crime trends, socio-economic influences, and predictive insights that can support policy-making and public safety research.
